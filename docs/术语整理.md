## 二、关节与运动方向

| 英文术语 | 中文翻译 | 说明 |
|----------|----------|------|
| DOF (Degree of Freedom)       | 自由度 | 机器人可独立运动的维度数量，本项目为 9 DOF |
| Revolute Joint | 旋转关节      | 绕单轴旋转的关节类型，本机器人所有关节均为此类型 |
| Fixed Joint    | 固定关节      | 不可动的连接（如 base_link 与 torso 之间） |
| Roll           | 横滚 / 侧倾   | 绕前后轴（X轴）旋转，身体左右摇摆 |
| Pitch          | 俯仰 / 前后倾 | 绕左右轴（Y轴）旋转，身体前后倾斜 |
| Yaw            | 偏航 / 航向   | 绕垂直轴（Z轴）旋转，身体左右转向 |
| Hip Roll       | 髋关节侧摆    | 控制腿的左右张开/收拢（X轴旋转） |
| Hip Pitch      | 髋关节俯仰    | 控制腿的前后摆动（Y轴旋转） |
| Ankle Pitch    | 踝关节俯仰    | 控制脚的前后翘/压（Y轴旋转） |
| Head Pitch     | 头部俯仰      | 控制头部上下点头 |
| Joint Limit    | 关节限位      | 关节允许的最大/最小旋转角度 |
| Joint Angle    | 关节角度      | 关节当前的旋转角度 |
| Joint Torque   | 关节力矩      | 关节电机施加的旋转力 |
| Joint Velocity | 关节角速度    | 关节旋转的速度 |

## 三、运动学与动力学

| 英文术语                 |    中文翻译   |            说明          |
|-------------------------|-------------|------------------------ |
| Kinematics              | 运动学       | 研究运动的几何关系，不考虑力 |
| Dynamics                | 动力学       | 研究力与运动的关系 |
| Forward Kinematics (FK) | 正运动学     | 已知关节角度 → 计算末端位置 |
| Inverse Kinematics (IK) | 逆运动学     | 已知末端位置 → 反算关节角度 |
| 2-Link IK               | 二连杆逆运动学| 针对两段连杆的解析求解方法（本项目用于大腿+小腿） |
| Analytical Solution     | 解析解       | 用数学公式直接求得精确解（区别于数值迭代法） |
| Center of Mass (COM)    | 质心 / 重心  | 物体质量的等效集中点 |
| ZMP (Zero Moment Point) | 零力矩点     | 地面上惯性力矩为零的点，用于判断行走稳定性 |
| Support Polygon         | 支撑多边形   | 所有接地点围成的区域，重心投影需在其内才稳定 |
| Moment / Torque         | 力矩        | 使物体旋转的力，等于力×力臂 |
| Inertia                 | 惯性/转动惯量 | 物体抵抗旋转加速度的量度 |
| Inertia Tensor          |惯量张量      | 用 3×3 矩阵描述物体绕各轴的转动惯量（ixx, iyy, izz 等） |
| Mass Distribution       | 质量分布     | 各部件的质量分配方案 |
| Gravity Compensation    | 重力补偿     | 施加力矩来抵消重力对关节的影响 |
| Sagittal Plane          | 矢状面       | 将身体分为左右两半的平面（前后方向运动在此平面内） |
| Frontal / Coronal Plane | 冠状面       | 将身体分为前后两半的平面（左右侧倾在此平面内） |

## 四、控制理论

| 英文术语 | 中文翻译 | 说明 |
|----------|----------|------|
| PID Controller | PID 控制器 | 比例-积分-微分控制器，经典反馈控制算法 |
| Proportional (P / Kp) | 比例项 | 校正量与误差成正比，决定响应速度 |
| Integral (I / Ki) | 积分项 | 消除稳态误差，但可能导致超调 |
| Derivative (D / Kd) | 微分项 | 预测误差变化趋势，起阻尼作用 |
| PID Gains | PID 增益 | Kp、Ki、Kd 三个参数的总称 |
| Integral Windup | 积分饱和 | 积分项过度累积导致控制输出过大的现象 |
| Integral Limit / Anti-Windup | 积分限幅 / 抗饱和 | 限制积分项的累积范围，防止 windup |
| Setpoint / Target | 设定值 / 目标值 | 控制器希望达到的目标（如目标高度 0.24m） |
| Error | 误差 | 设定值与实际值之间的差 |
| Correction | 校正量 | 控制器输出的调整量 |
| Feedback | 反馈 | 将输出状态返回给控制器，形成闭环 |
| Closed-Loop Control | 闭环控制 | 有传感器反馈的控制方式 |
| Open-Loop Control | 开环控制 | 无反馈的控制方式（只发指令不检查结果） |
| Position Control | 位置控制 | 设定目标角度，电机自动跟踪到达 |
| Velocity Control | 速度控制 | 设定目标速度，电机保持该速度 |
| Torque Control | 力矩控制 | 直接设定电机输出力矩 |
| Saturation / Clipping | 限幅 / 截断 | 将输出值限制在允许范围内 |
| Overshoot | 超调 | 响应超过设定值的现象 |
| Oscillation | 振荡 | 输出在设定值附近反复摆动 |
| Damping | 阻尼 | 抑制振荡的效果 |
| Steady State | 稳态 | 系统不再变化的最终状态 |
| Ziegler-Nichols Method | 齐格勒-尼科尔斯法 | 一种经典的 PID 参数整定方法 |

## 十二、坐标系与方向约定

本项目采用的坐标系约定（与 PyBullet 默认一致）：

| 轴 | 方向 | 对应旋转 |
|----|------|----------|
| **X 轴** | 前方为正 | Roll（横滚）绕此轴旋转 |
| **Y 轴** | 左方为正 | Pitch（俯仰）绕此轴旋转 |
| **Z 轴** | 上方为正 | Yaw（偏航）绕此轴旋转 |

常见方向描述：
- **Forward** = +X 方向（前进）
- **Lateral** = Y 方向（侧向）
- **Vertical** = Z 方向（垂直）
